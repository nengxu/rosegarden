Rosegarden-4 for RedHat

The latest Rosegarden-4 release is available as a RedHat RPM from Planet CCRMA.

To initially install Rosegarden-4, a PlanetCCRMA user needs only to type:

apt-get update
apt-get install rosegarden

Rosegarden-4 for Debian

Rosegarden-4 is available as a package from Debian Unstable (thanks to Enrique Robledo Arnuncio).

Rosegarden-4 for your distribution

Rosegarden-4 should hopefully be included as part of the next SuSE release (8.2) but we're notably still missing from Mandrake. If we're not packaged as part of your favourite distribution then please get in touch with your distribution manufacturer and ask why not! If you've made an RPM or package for any distribution then please let us know so that we can link it here.

If you'd like to support Rosegarden-4 but you aren't a developer, another way to contribute is with a financial donation.
A professional sequencer costs hundreds of pounds. We haven't written one yet, but we aren't asking for hundreds of pounds either. Twenty quid or so would be great.
Thanks!

Rosegarden-4 source code

If you can't find a package to suit your system them you can still build Rosegarden-4 from source code. The latest release is shown below:

  • Rosegarden-4 release 0.8, 4 October 2002: get the sources from the SourceForge page.

You can also obtain the most recent source code (which might not work) directly from the CVS repository. The SourceForge site also contains some code documentation and a mailing-list.

See the requirements page for details on how to build the source code.

Design reports

Rosegarden started life as a university project. These are the original reports Chris and Andy produced for that project, way back in the first half of 1994. They probably still constitute the best description of the internals of the application, although many details are now out-of-date.

  • Notation editor design document (212K, gzipped PostScript).
  • Sequencer design document (691K, Acrobat PDF).
  • Interlock tool-communication framework design document (96K, Acrobat PDF).
