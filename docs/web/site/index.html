What is Rosegarden-4?

Rosegarden-4 is an attractive, user-friendly audio and MIDI sequencer, score editor, and general-purpose music composition and editing application for Unix and Linux. It is currently approaching 1.0 feature completion and beta testing following two and a half years of sustained active development.

Current Release

The latest release of Rosegarden is Version 4, Release 0.9 released on 29th April 2003. Have a look at some pictures of it in use, download it, or read the Frequently Asked Questions page.

This is still a pre-beta release, but it is entirely usable by interested users and developers. It features complete or partial support for:

  • Score, piano-roll and track overview editors
  • Score interpretation of performance MIDI data
  • MIDI and audio playback and recording using ALSA and JACK
  • Audio plugin support using LADSPA
  • MIDI file I/O and Csound export
  • High-quality score printing by exporting Lilypond files
  • Translations for Russian, Spanish and German
  • Shareable Device (.rgd) files to ease MIDI portability

Changes since the previous Rosegarden-4-0.8.5 release include:

  • Significant improvements in rendering and editing notation from performance MIDI data
  • Ability to quantize for notation without changing performance
  • Performance interpretation of dynamics and accent stresses
  • Corrected display of all accidentals in (or against) every key
  • Marks and Group toolbars
  • Bank Editor dialog reworked - many fixes and new apply/reset methodology. Device "librarians" - increased number of pre-supported patchsets.
  • Split-by-pitch with ranging split point
  • Tip of the Day
  • "Revert file" option
  • Studio-> Save Current Document as Default Studio to make short work of saving your own autoload.rg
  • Tracks toolbar
  • Detailed sequencer status information available in Configure dialog
  • Document autosaving and recovery
  • More complete Right Mouse Button menu in tracks canvas
  • Document loading rework
  • Better progress reporting for long operations
  • Segment labelling
  • Floating value help when moving rotaries/sliders
  • Tracks can be moved up and down in the Composition [buggy]
  • Better handling of the sequencer process
  • Lots of translations - Russian, Spanish, French, German
  • Rosegarden Tutorial (including a PDF) thanks to Michael McIntyre
  • A lot of audio improvments - stereo recording, panning, audio faders.
  • MIDI synchronisation infrastructure
  • ALSA client dynamic detection. ALSA recording device selectin.
  • Import of Banks/Programs from .rgd files and standard Soundfont (sf2) files
  • Library of premapped MIDI devices in .rgd files
  • Many Lilypond export improvements

Rosegarden-4 is Free Software under the GNU GPL. It is a KDE application, which means to build it you need the KDE 3.x libraries. However, once installed it can be run within the KDE environment or any other desktop or window manager environment.

Note: if you have a Linux distribution that includes a package simply called rosegarden, you might have Rosegarden-2.1. (We always recommend that Rosegarden-4 is packaged as rosegarden4.) Consider upgrading!

Rosegarden-2.1

Rosegarden-2.1 is a quite separate and much older program, containing a reasonable score editor and a reasonably useless sequencer with limited integration between the two. It's very useful for certain tasks, particularly on older hardware, but Rosegarden-4 is where all the fancy new stuff is at.
