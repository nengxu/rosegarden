What is Rosegarden-4?

Rosegarden-4 is an attractive, user-friendly MIDI and audio sequencer, notation editor, and general-purpose music composition and editing application for Unix and Linux. It is currently somewhere approaching beta quality, following two years of sustained active development.

Rosegarden-4 is Free Software under the GNU GPL. It is a KDE application, which means to build it you need the KDE 2.2.x or 3.0.x libraries. However, once installed it can be run within the KDE environment or any other desktop or window manager environment.

Note: if you have a Linux distribution that includes a package simply called rosegarden, you almost certainly have Rosegarden-2.1. (We always recommend that Rosegarden-4 is packaged as rosegarden4.) Consider upgrading!

Current Release

The latest release of Rosegarden is Version 4, Release 0.8, released on 4th October 2002. Have a look at some pictures of it in use, download it, or read the Frequently Asked Questions page.

This is still an early, development release, but it is entirely usable by interested users and developers. It features complete or partial support for:

  • Score, piano-roll and track overview editors
  • MIDI and audio playback and recording using ALSA and JACK
  • Audio plugin support using LADSPA
  • MIDI file I/O and Csound export
  • High-quality score printing by exporting Lilypond files

Changes since the previous 0.2.0 release include:

  • Improved Notation Printing (draft quality)
  • realtime LADSPA plugin support (chainable plugins for each Instrument)
  • Note entry using the PC keyboard in notation view
  • Major Matrix view improvements (zooming, snapgrid size, quantizing)
  • Major Performance improvements in sequencer
  • "Stuck notes" problem fixed
  • Matrix velocity setting dialog
  • Segment positioning guides
  • Turn repeating Segments to real copies
  • Matrix view auto scrolling
  • Rudimentary lyric editor
  • Chorus, Reverb, Resonance, Filter, Attack, Release MIDI controls
  • Specialised Rotary Control widget
  • Repeating Segments work to end of Composition marker
  • Composition Start and End Markers can be changed
  • Splitting Segments fixed
  • Audio volume faders
  • EventList editor extended
  • Better Audio Waveform Previews
  • SysEx, Program Changes, Controller Changes, PitchBend all supported
  • and editeable at Composition level
  • Toolbars & Settings saving fixed
  • Some new useful keyboard shortcuts
  • Optional Transport toolbars
  • More preference settings
  • A couple of new note transformation functions
  • Some fixes to progress reporting
  • More help text for Notation

Rosegarden-2.1

Rosegarden-2.1 is a quite separate and much older program, containing a reasonable score editor and a reasonably useless sequencer with limited integration between the two. It's very useful for certain tasks, particularly on older hardware, but Rosegarden-4 is where all the fancy new stuff is at.
