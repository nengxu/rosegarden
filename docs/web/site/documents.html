User Documentation for Rosegarden-4

Here's the beginning's of a Rosegarden Tutorial. It is updated regularly by Rosegarden-4 users so have a look to see what's in there and give your feedback on the Rosegarden mailing lists.

For those of you looking to create voice mappings for your own MIDI hardware, here's a mini-HOWTO for creating voice definitions for Rosegarden-4. You should also check out the ever-growing device library (a snapshot of which is included with each Rosegarden-4 release) to make sure your device hasn't already been mapped.

And for anyone interested in helping out with translations, here's a translation mini-HOWTO.

Developer Documentation for Rosegarden-4

Automatically generated class hierarchies and extracted code notes can be found on our daily Doxygen dump. Useful for reference and as an introduction to some parts of the code.

There are also some notes to be found in Rosegarden CVS itself.

General MIDI and Audio information

For general information on how to set up your home studio you can consult Bownie's DIY Guide to Recording. It's a bit Windows-centric still but it has some useful information for beginners to the world of home recording.
